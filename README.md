# Tupack

Tool for put Tizen.Tracer functions to methods.

```
TuPack:
  Application for implant Tizen Trace to DLL, use ";" for sperator of list

Usage:
  TuPack [options] [<args>...]

Arguments:
  <args>    paths of DLL

Options:
  --namespaces <namespaces>    target namespace list
  --types <types>              target type list
  --methods <methods>          target method list
  --api <api>                  Tizen.NET API version
  --repack                     Repackaging if a directory of first dll is child of tpk root
  --version                    Display version information

```
